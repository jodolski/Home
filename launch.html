Launch day

May 7th, 2020

It's finally here. A milestone reached. For the first time in a long time, I have a new website. No, it's not some landing page or simple portfolio, but a place where I can call home. It's got everything that I wanted to share: my professional work, personal journey, photography, and writings.

The overall visual design was decided on a couple of principles. Dark mode is the default state (I'll explore light mode toggling in the future). The layout should be standard. Navigating through sections to get information (for all of you recruiters!) should be clear. One of the ways to let users know where they are is to embolden the parent section they're on. For example, if a user explores one of the case studies, the user will know that they're under the Work parent section.

The gratituous use of orange throughout the website is inspired by the colors of the desert. The footer is a hand drawn silhoutte of Black Mountain in Nevada which I used to hike hundreds of times before moving to Austin. The beauty that brings it all in together: the custom CSS gradient background that reminds me of the stunning sunsets you'd see across the Las Vegas Valley.

Cutting straight to the technicals: this website is written from scratch in pure HTML and Tailwind CSS (a utility-first framework for stylizing). It's set up to be responsive to work across mobile and desktop devices (if not, do let me know). The code is pushed to GitHub and hosted by Netlify. The domain name was purchased through Porkbun of which I've experienced nothing but excellent customer service. I've yet to choose a static site generator but I'm looking at Gatsby as a strong possibility. And finally, this website has been vetted for accessibility using Axe.
